# Caprica Brand Style Guide

## Logo
The Caprica logo is a military-inspired shield with a gold "C" and "Caprica" text. It is implemented as a transparent PNG for high-resolution display.

- **Primary Logo**: `/caprica-logo.png`
- **Favicon**: `/favicon.png`

## Color Palette

### Interface Colors
| Role | Color | Hex | Usage |
|------|-------|-----|-------|
| **Header / Banner** | Metallic Slate | `#cbd5e1` | Navigation bar background (Ensures navy logo visibility) |
| **Brand Navy** | Logo Navy | `#122f4f` | Sampled from logo, used for text and accents |
| **Brand Gold** | Gold | `#cd922a` | Sampled from logo, used for highlights |
| **Background** | Light Gray | `#f8fafc` | Page backgrounds |
| **Surface** | White | `#ffffff` | Cards and content areas |

### Typography
- **Primary Font**: Inter (Sans-serif)
- **Monospace**: JetBrains Mono (For technical/military data)

## Implementation Details
- **Theme**: Professional Light Theme
- **Header**: Metallic finish with Navy text for distinct military-industrial aesthetic and high readability.
